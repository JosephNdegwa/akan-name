# AKAN-NAMES (Ghana)

## Description
This is a web-page that gives the user an Akan name (a Ghanian tribe) according to the day of the week (s)he was born.

### Author
Joseph Ndegwa

## Setup and installation
- Run `git clone https://github.com/JosephNdegwa/akan-name.git`
- Serve on browser
